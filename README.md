# 🎬 Netflix India Landing Page Clone

A pixel-perfect front-end clone of the Netflix India landing page, built using modern responsive layout techniques. This project focuses on replicating premium UI details, exact typographic scales, custom interactive elements, and custom-tailored cross-browser form inputs.


---

## ✨ Features Built

*   **Dynamic Hero Area:** Replicated the iconic Netflix ellipse background mask with custom multi-layer linear gradients (fading to solid black at the top and signature Netflix Red at the bottom).
*   **Custom Form Optimization:** Optimized email sign-up wrappers featuring fine-tuned CSS Flexbox layout values (`flex: 1.5`) for a sleek, compact input-to-button size proportion on desktop displays.
*   **Trending Slider Row:** Features high-resolution asset displays complete with Netflix’s distinctive high-contrast outline rank numbering (`-webkit-text-stroke` layout layer overrides).
*   **CSS Grid Implementation:** Built a fully dynamic "More reasons to join" section utilizing `grid-template-columns: repeat(auto-fit, minmax(280px, 1fr))` for fluid card redistribution across mobile, tablet, and widescreen viewports.
*   **Interactive Micro-Animations:** Clean `:hover` transition scaling, font-awesome typography tracking, and a fully custom-aligned international language dropdown component embedded with clean flex vector icons (`bi-globe`).

---

## 🛠️ Built With

*   **HTML5** – Semantically structured DOM nodes.
*   **CSS3** – Flexbox, Grid, custom pseudo-elements (`::after`), and hardware-accelerated transforms.
*   **Bootstrap Icons** – Integrated inline vector assets for fluid UI interaction scaling.
*   **Official Netflix Brand Assets** – Sourced direct high-resolution vector favicons and official logos straight from production CDN networks.
